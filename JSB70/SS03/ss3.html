CSS!

(Cascading Style Sheets)

Selectors:

Tag selector

Selects all tags of the selector

                p {
                    color: rgb(0,0,0);
                    background-color: rgb(255,255,255);
                }

Child selector

Selects all tags under a specific tag

                div p{
                    text-align: center;
                }

Direct-child selector

Selects ONLY the tags that are direct child of a specific tag

                div > p{
                    border: 2px;
                }

Types of CSS

Inline

                <p style=""></p>

Internal

                <head>
                    <style>
                        ....
                        a little bit of css later
                        ....
                    </style>
                </head>

External

                ---main.html---

                <head>
                    <link rel="stylesheet" href="style.css">
                </head>

                ---style.css---

                p {
                    what p tag looks like, but in css
                }
