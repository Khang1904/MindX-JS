CSS priority

Inline           ˄ High
Internal        ||
External       || Low

Selectors pt.2

(Goes High to Low priority)

                /* ID selector (applies to id) */
                #id{
                    /* css here */
                }

                /* Class selector (applies to class) */
                .class{
                    /* css here */
                }

                /* Element selector (applies to tag) */
                p {
                    /* css here */
                }

                /* Universal selector (applies to EVERYTHING) */
                * {
                    /* css here */
                }

Ways to add CSS:

(..still High to Low priority)

                <-- Inline -->
                <p style=""></p>

                <-- Internal -->
                <style>
                    /* css here */
                </style>

                <-- External -->

                In the html file:
                <link rel="stylesheet" href="*link to your file*>

                then do the styling over there!
